# 7hBee-Movie-Texter
> Texts the complete Bee Movie Script line by line
---
# Installation
1. just clone the Repository
```shell
git clone https://github.com/7h3730B/7hBee-Movie-Texter.git
```
2. cd into it
```shell
cd 7hBee-Movie-Texter/
```
3. Compile it with g++ or any other c++ compiler
```shell
g++ main.cpp
```
4. Run it
```shell
./a
```
5. Profit!!?!
